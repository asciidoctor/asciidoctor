Asciidoctor works on Linux, macOS and Windows.

Asciidoctor requires one of the following implementations of Ruby:

* Ruby 1.8.7
* Ruby 1.9.3
* Ruby 2 (2.0.0 or better)
* JRuby 1.7 (Ruby 1.8 and 1.9 modes)
* JRuby 9000
* Rubinius 2.0 (Ruby 1.8 and 1.9 modes)
* Opal (Javascript)
