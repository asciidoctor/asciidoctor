The benefit of installing Asciidoctor using a Linux package manager is that the manager will also install Ruby and RubyGems if they're not already on your machine.

== dnf

//Fedora or RHEL

To install Asciidoctor on Fedora or RHEL via EPEL using the {url-fedora}[rubygem-asciidoctor^] package:

. Open a terminal
. Run the installation command on Fedora:
+
 $ sudo dnf install asciidoctor

include::{partialsdir}/success.adoc[]

== apt-get

//Debian or Ubuntu

To install Asciidoctor on Debian or Ubuntu via `apt-get`:

. Open a terminal
. Type the following `apt-get` command using sudo:

 $ sudo apt-get install asciidoctor

include::{partialsdir}/success.adoc[]

== apk
// Alpine Linux

To install Asciidoctor on Alpine Linux using the {url-alpine}[asciidoctor package^]:

. Open a terminal
. Type the following `apk` command using sudo:

 $ sudo apk add asciidoctor

include::{partialsdir}/success.adoc[]

== gem install

To install Asciidoctor using the `gem` command:

. Open a terminal and type:

 $ gem install asciidoctor

include::{partialsdir}/success.adoc[]

== Install a pre-release version

To install a pre-release version of Asciidoctor:

. Open a terminal and type:

 $ gem install asciidoctor --pre

== Install using Bundler

To install Asciidoctor for a project using Bundler:

. Open your project's Gemfile
. Add the `asciidoctor` gem using:

 gem 'asciidoctor'

. Save the Gemfile
. Open a terminal
. Install the gem using the `bundle` command:

 $ bundle
