|===
|Attribute |Value(s) |Example Syntax |Notes

|sectids
|N/A
|`:sectids!:`
|Set to autogenerate section IDs by default

|idprefix
|`_`, or user defined text
|`:idprefix: id_`
|Set to prepend string to generated section ID

|idseparator
|`_`, or user defined character
|`:idseparator: -`
|Set to insert character between words in generated section ID

|options; %
|`untitled`
|`[colophon%untitled]`
|DocBook only; can only be set on special sections
//Asciidoctor only

|sectanchors
|N/A
|`:sectanchors:`
|
//Asciidoctor only

|sectlinks
|N/A
|`:sectlinks:`
|
//Asciidoctor only

|sectnums
|N/A
|`:sectnums:`
|Off by default;
can be toggled on or off in the flow of the document
// replaces numbered in AsciiDoc.py

|sectnumlevels
|0-5
|`:sectnumlevels: 3`
|Autogenerate section numbers up to level three by default;
value of `0` is the same as `sectnums!`
|===
