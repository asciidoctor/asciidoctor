== Lead style
// para-lead.adoc, included in: user-manual: paragraphs: lead

Apply the `lead` style to any paragraph, and it will render using a larger font size.
The lead style is assigned to the xref:attributes:role.adoc[role attribute].
You can set `role` using the long- or shorthand method.
