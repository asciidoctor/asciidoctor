Asciidoctor operates on macOS, Windows, and most Linux distributions.

== Supported Platforms

Platform support is specific to each major Asciidoctor version.

[NOTE]
.Linux Distributions
====
While we only test Asciidoctor on Alpine, Debian, and Fedora, it should operate as expected on most Linux distributions.
